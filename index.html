Flags

An assortment of SVG flags and associated projects

Most of these flags were sourced from Wikipedia. Many of them have been improved by me, e.g. cleaning up and modernizing the SVG XML, fixing width/height issues, and adding explicit fill attributes to improve ease of color parsing. A few were created by me either from scratch, using vtracer, or by borrowing bits and pieces of SVG XML from other flags.

  • Flags Game - A vexillology guessing game using k-means clustering to make it very challenging
  • Flag Clusters - A listing of the flag clusters used in flags game
  • All Flags - Listing of all flags, filterable using the URL arguments lang= and tags=
